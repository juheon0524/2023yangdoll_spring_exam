응용 프로그램의 동작 원리(WHP)
1. 운용체제에서 바탕화면에 hwp아이콘을 더블클릭
=> 운영체제가 main method를 호출

2. 운영체제는 hwp 프로그램에게 메모리 할당을 해준다.
=> 운영체제에 의해서 메모리를 할당 받는다.(크게 2개의 영역으로 나눈 후 스택영역에 실행 메서드공간을 확보한다.)

3.자료를 보관(Hwp에서 문서를 편집) => 변수를 만들고(함수에서 정의되어 있는 모든 변수를 순서에 따라서 만들고 자료를 넣거나 읽거나...)
  변수를 만들떄 반드시 크기가 결정된다.
  변수 생성 1순위 => 매개변수
