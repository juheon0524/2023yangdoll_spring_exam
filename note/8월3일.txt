JDK : 개발환경 (JRE포함)

JRE => 자바 실행환경

인간 중심의 언어 (Text code)  기계어(target,바이너리코드)
변환 ==> 한꺼번에 (compiler : 속도가 빠름, 하드웨어에 종속)  한줄씩(Interprinter : 속도가 느림, 종속석이 낮다.)

클래스 소속된 함수= 메서드

반드시 변수는 선언후 사용  --> (타입)변수  

큰타입은 작은 타입을 담을 수 있다. 
큰타입을 작은타입에 넣을떄는 데이터 손실을 감수해야한다.

extends => 확장
상속(같은 것 끼리) 클래스와 클래스간에 계층구조, 인터페이스와 인터페이스간에 계층구조 => 단일
implements => 구현 
구현(다른 것 끼리) => 다중

변수의 타입은 실제 저장된 자료를 어떻게 해석 해야 하는지 알려주는 정보이다.

double
float
long 
int

캡슐화 상속 추상화 다형성


필드만 있는 인터페이스를 만드는 것을 한번쯤 고려하라!
